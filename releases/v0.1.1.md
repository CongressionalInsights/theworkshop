# TheWorkshop `v0.1.1` - Provider-Agnostic Image Credentials for OSS

TheWorkshop `v0.1.1` completes the open-source credential portability baseline for image generation.

## Highlights

- Added canonical image credential flow for image generation:
  - `THEWORKSHOP_IMAGEGEN_API_KEY` (preferred, cross-platform, env-first)
- Retained compatibility aliases for existing local setups:
  - `OPENAI_API_KEY`
  - `OPENAI_KEY`
- Added optional keychain provider mode:
  - `THEWORKSHOP_IMAGEGEN_CREDENTIAL_SOURCE=keychain`
  - optional `$apple-keychain` fallback integration with `THEWORKSHOP_KEYCHAIN_RUNNER` / service overrides
- Added explicit provider override support in `scripts/imagegen_job.py` and improved provider diagnostics.
- Updated preflight credential checks in `scripts/doctor.py` to pass on env-only setups.
- Added tests covering env success, env-only failure guidance, and provider-aware preflight behavior.
